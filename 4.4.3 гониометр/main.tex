\begin{center}
В работе исследовано использование призмы в качестве спектрального прибора. При помощи гониометра измерен полученный в призме из тяжелого флинта спектр излучения ртути. По нему рассчитана дисперсия коэффициента преломления стекла, из которого изготовлена призма, определена марка стекла. Проведена оценка спектральных характеристик призмы: угловой дисперсии и разрешающей способности.
\end{center}

\section{Теоретическое введение}

\begin{wrapfigure}{r}{0.3\textwidth}
  \begin{center}
    \caption{Ход лучей в призме для угла наименьшего отклонения}
    \includegraphics[width=0.25\textwidth]{pics/refraction}
    \label{fig:refraction}
  \end{center}
\end{wrapfigure}

Показатель преломления материала призмы $n(\lambda)$ удобно определять
по углу наименьшего отклонения $\theta(\lambda)$ (рис. \ref{fig:refraction}). Угол минимального отклонения $\theta$, преломляющий угол $\alpha$ и показатель преломления
связаны соотношением:

\begin{equation}
    n(\lambda) = \frac{\sin(\frac{\alpha + \theta(\lambda)}{2})}{\sin{\frac{\lambda}{2}}}
    \label{eq:refr_ind}
\end{equation}

Угловая дисперсия призмы в окрестности угла наименьшего отклонения определяется по формуле:

\begin{equation}
    D_\theta \equiv \frac{\delta\phi}{\delta\lambda} = \frac{l}{a} \frac{dn}{d\lambda} =\frac{2 \sin\frac{\alpha}{2}}{\cos\theta} \frac{dn}{d\lambda} = \frac{2 \sin\frac{\alpha}{2}}{\sqrt{1 - \sin^2\frac{\alpha}{2} n^2}} \frac{dn}{d\lambda}
    \label{eq:disp}
\end{equation}

, где $\frac{dn}{d\lambda} = D$ -- дисперсия коэффициента преломления, $n$ -- показатель преломления.

Разрешающая способность призмы определяется только эффективной длиной основания:

\begin{equation}
    R \equiv \frac{\lambda}{\delta \lambda} = l \left|\frac{dn}{d\lambda}\right|
    \label{eq:R}
\end{equation}

При помощи уголкового отражателя и стеклянной пластины была проведена юстировка гониометра Г5. Дополнительно прибор был отъюстирован под исследуемую призму.

По отражению автоколлимационного креста был найден преломляющий угол призмы $\alpha_0 = 62^\circ23'30'' \pm 10''$.

Далее перед щелью коллиматора была установлена ртутная лампа. Преломленные призмой лучи от лампы наблюдались через зрительную трубу гониометра (рис. \ref{fig:beauty}). Для каждой линии спектра был измерен минимальный угол отклонения луча $\theta(\lambda)$ и по формуле \eqref{eq:refr_ind} рассчитан коэффициент преломления ($\epsilon_n \sim 0.01\%$). Результаты приведены в таблице \ref{tab:disp}.

По зависимости коэффициента преломления от длины волны (рис. \ref{fig:graph}) рассчитана средняя дисперсия показателя преломления стекла призмы в видимом диапазоне $\boxed{D = -1.3\cdot 10^3 ~\text{см}^{-1}}$ ($\epsilon_D \sim 0.1\%$). Это значение ближе всего к значению дисперсии коэффициента преломления для марки тяжелого флинта ТФ1 ($D_{\text{ТФ1}} = -1.9\cdot 10^3 ~\text{см}^{-1}$).

\section{Оценка спектральных характеристик призмы}

\subsection{Угловая дисперсия}

Для расчета угловой дисперсии, при неизменном угле падения пучка на призму, было измерено угловое расстояние между желтыми ($\lambda_1 = 577,0$ нм и $\lambda_2 = 579,1$ нм) линиями ртути $\delta \phi = 1'15'' \pm 7''$. Отсюда угловая дисперсия $\boxed{D_{\theta\text{, exp}} = \frac{\delta\phi}{\delta\lambda} = -(35 \pm 3)'/\text{нм}}$, что сопоставимо с угловой дисперсией дифракционной решетки с периодом $d \sim 6$ мкм в первом порядке ($D_\text{реш} \sim 1/d$).

Теоретическое значение угловой дисперсии из формулы \eqref{eq:disp} составляет \\$\boxed{D_{\theta\text{, th}} = -(30,3 \pm 0,7)'/\text{нм}}$. \ul{Экспериментальное и теоретическое значения совпадают в пределах $2\sigma$}.

\subsection{Разрешающая способность}
Минимально возможная для наблюдения угловая полуширина желтой линии ртути ($\lambda_1 = 577.0$ нм) составила $\delta\theta = (15 \pm 5)'$. Отсюда разрешающая способность оптической системы гониометр-призма-глаз в окрестности желтого дублета равна $\boxed{R = \frac{\lambda_1 D_{\theta, \text{exp}}}{ \delta \theta} = 1300 \pm 500}$. Для достижения сопоставимой разрешающей способности в первом порядке дифракционной решетки с периодом $d = 10$ мкм потребовался бы пучок шириной $a = R d \sim 1$ см.

С использованием формулы \eqref{eq:R} по полученному значению разрешающей способности можно вычислить эффективную длину основания призмы $\boxed{l_{\text{эфф}} = \frac{R}{D_\text{желт}} = (1.5 \pm 0.5) ~ \text{см}}$ и ширину пучка на входе в призму $\boxed{a = \frac{\cos\theta_\text{желт}}{2\sin\frac{\alpha}{2}} l_\text{эфф} = (0.9 \pm 0.3)~\text{см}}$. 

Если бы в формировании изображения спектра был задействован весь объем призмы (длина основания $l = (7.2\pm0.1) см$), то разрешающая способность равнялась бы $\boxed{R_max = l D = 6300 \pm 200}$.

\section{Заключение}
Проделанная работа демонстрирует состоятельность призмы как спектрального прибора: ее угловая дисперсия и разрешающая способность оказались в эксперементе сопоставимыми с аналогичными характеристиками для дифракционной решетки с периодом $\sim 10$ мкм в первом порядке. 

По спектру ртути с точностью $0.1\%$ была определена дисперсия показалеля преломления стекла призмы $D = -1.3\cdot 10^3 ~\text{см}^{-1}$.
