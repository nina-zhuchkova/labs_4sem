\subsection{ Определение ширины щели по увеличенному изображению}

Первый метод измерения ширины щели заключался в измерении ширины ее увеличенного при помощи короткофокусной линзы изображения (см. рис. \ref{fig:A1_setup}).
\begin{figure}[h]
    \centering
    \includegraphics[width = 0.9\textwidth]{pics/A1_setup.jpg}
    \caption{Схема для определения ширины щели с помощью линзы}
    \label{fig:A1_setup}
\end{figure}

При измерениях расстояние от щели до экрана было равно $L \equiv a_1 + b_1 = (136 \pm 0.5)$ см, а расстояние от линзы до экрана $b_1 = (131 \pm 0.5)$ см. Увеличение может быть посчитано двумя способами:
\begin{itemize}
    \item Напрямую из измеренных расстояний: $\Gamma = \frac{b_1}{L - b_1} = 26 \pm 4$ 
    \item Через формулу тонкой линзы: $\Gamma = \frac{F_2 b_1}{b_1 - F} = 33,5 \pm 0,2$
\end{itemize}

Несмотря на б\'oльшую погрешность, далее в расчетах использовалось увеличение, посчитанное первым методом: он лучше учитывает погрешность, связанную с неточной фокусировкой изображения на экране.
